E2: Project documentation

The purpose of this documentation is to aid in the comprehension of the source code of the E2 project. It provides a description of ideas behind implemented algorithms and data structures. The source code itself (especially header files) then contains important Doxygen [1], comments which clarify concrete implementation details of the ideas.

Structure of the project

Modules:

cellab: A topological structure of a neural tissue and simulation algorithm of the tissue.
cellconnect: Algorithm for creating initial interconnection of cells in a neural tissue.
efloop: Interconnection between one or more neural tissues and an environment.
envlab: Creation of environments.
utilities: General programming utilities used throughout other modules.
test suite: A test suite for all modules above.
